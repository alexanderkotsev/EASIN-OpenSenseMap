EASIN 
=========
This is the front-end for [EASIN - Eauropean Alien Species Information Network](http://easin.jrc.ec.europa.eu/).


### Installation

1. Install node.js 
2. Install npm with the following command:

```
npm install
```

3. Install bower components with the following command:

```
bower install
```
